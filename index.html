📖 How to Use

  1. Press F6 and hover over each browser window you want to add
  2. Click button "➕ เพิ่มจอ" or press F6 to register the window
  3. Type your messages in the text editor (one message per line)
  4. Click "🚀 ยิงข้อความ" or press F1 to broadcast
  5. Click "🎯 จัดหน้าจอ" to auto-arrange windows in grid layout
  6. Use "↻" button to refresh network information
